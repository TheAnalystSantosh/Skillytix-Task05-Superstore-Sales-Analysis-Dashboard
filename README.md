
# Superstore Sales Analysis Dashboard – Power BI Project

## Overview
This project analyzes the Superstore dataset using Power BI to extract insights on sales,
profit, customer segments, and regions.

## Dataset
- Orders
- Sales
- Profit
- Discount
- Customer & Product details

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Features
- KPI Cards (Sales, Profit, Orders, Margin)
- Sales Trend by Month
- Category & Product Insights
- Regional Map
- Customer Segmentation
- Interactive Slicers
